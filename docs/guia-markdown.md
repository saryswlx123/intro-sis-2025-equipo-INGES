# Guía básica de Markdown  
Markdown es un lenguaje de marcado ligero que permite dar formato a textos de manera sencilla. A continuación se presentan algunos ejemplos básicos.  

## 1. Títulos  
Se utilizan `#` para crear títulos.  

```markdown
# Título nivel 1
## Título nivel 2
### Título nivel 3
#### Título nivel 4
